# spring.io-replica
Basic static HTML/CSS project that recreates key sections of the Spring.io homepage for layout and styling practice.

## Project Structure
- `index.html`
- `index.css`
- Images (`.png` assets) in the same folder

## Run Locally
Open `index.html` directly in a browser, or start a simple local server:

```bash
python -m http.server 8080
```

Then visit `http://localhost:8080`.

## Notes
- If you use VS Code Live Server, it usually runs on `http://localhost:5500`.
